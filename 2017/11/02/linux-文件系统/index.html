linux 文件系统

文件系统分类等。还是什么也不懂，就用ext4。

文件系统是用于控制所有程序在不适用数据时，这些数据如何存储，如果需要数据时，如何访问，还有元数据信息。

windows和mac的文件系统为NTFS和HFS+。

ext4文件系统

linux操作系统的现在默认广泛使用的是ext4，它是由ext3改进来的，ext3是由ext2改进来的。
是一个日志文件系统，会对文件在磁盘中的位置和磁盘的更改做记录。不支持透明压缩、重复数据删除或者透明加密。技术上支持了快照，但是处于实验阶段。最大文件大小是16 tebibytes。

btrf文件系统

又称BetterFS、Butter FS 或者 B-Tree FS。出现是原因是开发者希望扩展文件系统的功能，快照、池化、校验。最大文件比ext4大。

XFS 文件系统

扩展文件系统的一个扩展。RedHat 5.4以上支持。缺陷不能压缩，删除大量文件时性能底下。

总结

大概是什么也不懂就用EXT4吧。摊手。
