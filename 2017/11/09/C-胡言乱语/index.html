C++ 强制转换

4种

  1. static_cast(value)
    相互联系类型。常用。

  2. const_cast(value)
    转换成常量或者把const去掉

  3. reinterpret_cast
    转换任何内置数据

  4. dynamic_cast
    运行时完成的，不用于内置基本类型转换，成功返回指针失败NULL，基类一定要有虚函数。
