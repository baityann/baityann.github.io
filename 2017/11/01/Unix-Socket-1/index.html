Unix Socket 1

进程间通信方式，比socket更快，更稳定。

Unix domain socket是一种单机通信方式。用于同一台主机上两个进程间的通信。 socket本是网络通信， unix Socket 是在socket框架上发展出来的IPC机制。 Unix Socket 比socket更加有效率， 不经过网络协议，不打包拆包，只是将应用层数据从一个进程拷贝到另外一个进程。 Unix socket 也提供面向流和面向数据包两种API接口，但是是可靠的，消息不会丢失也不会错乱。
Unix socket 不是绑定一个端口，而是绑定一个文件。

参考[https://akaedu.github.io/book/ch37s04.html]
